## ¿QUÉ ES CLAUDE?

Claude es una IA conversacional que puede:

- ✅ Explicar conceptos de programación
- ✅ Generar y debuggear código
- ✅ Responder dudas complejas
- ✅ Mejorar tu código
- ✅ Crear ejemplos prácticos

**Gratis en:** https://claude.ai

**¿Por qué Claude y no ChatGPT?** Mejor para educación: explicaciones más claras, código limpio, responsable.

## 📚 TIPOS DE PREGUNTAS

### APRENDER CONCEPTO

```
"No entiendo [X]. ¿Puedes:
1. Explicar simple
2. Dar ejemplo real
3. Código en Python
4. ¿Cuándo se usa?
5. Errores comunes"
```

### GENERAR CÓDIGO

```
"Necesito función Python que:
- [Qué hace]
- Input: [parámetros]
- Output: [resultado]

Requisitos:
- [Validación]
- [Manejo errores]

Por favor: código comentado + ejemplo"
```

### DEBUGGEAR

```
"Error: [ERROR EXACTO]

Código:
\`\`\`
[TU CÓDIGO]
\`\`\`

¿Qué está mal?
¿Solución?"
```

### MEJORAR CÓDIGO

```
"¿Cómo mejoro?

\`\`\`
[TU CÓDIGO]
\`\`\`

Considera: eficiencia, legibilidad, prácticas"
```

## ✅ LO QUE SÍ HACER

- ✅ Sé específico en preguntas
- ✅ Proporciona contexto (Fase, nivel)
- ✅ Incluye código/error si es relevante
- ✅ Pide formato de respuesta
- ✅ Valida TODO lo que te proporciona
- ✅ Itera preguntas de seguimiento
- ✅ Comenta mucho el código
- ✅ Experimenta modificando

---

## ❌ LO QUE NO HACER

- ❌ Copiar sin entender
- ❌ Preguntas demasiado vagas
- ❌ No proporcionar contexto
- ❌ Confiar ciegamente sin validar
- ❌ Usar para evitar pensar
- ❌ Asumir que siempre está correcto
- ❌ Preguntas aisladas (conversa)
- ❌ Ignorar explicaciones

---
