## 💡 Conversación Ejemplo

### TÚ:

```
Hola, soy Fase 1 (Pseint).
¿Qué es una variable?
¿Por qué las necesito?
```

### CLAUDE:

```
Una variable es como una CAJA donde guardas datos.

ANALOGY:
- Variable = Caja
- Nombre = Etiqueta en la caja
- Valor = Lo que hay adentro

EJEMPLO:
nombre = "Juan"
edad = 25
precio = 19.99

"Juan" es texto
25 es número
19.99 es número decimal

Por qué las necesitas:
- Guardar información
- Usarla después
- Cambiarla cuando quieras
```

### TÚ:

```
¿Y si guardo varios valores?
¿Cómo hago?
```

### CLAUDE:

```
Usa una LISTA:

nombres = ["Juan", "María", "Pedro"]
números = [1, 2, 3, 4, 5]

Y accedes así:
nombres[0]  = "Juan"
nombres[1]  = "María"
```

## ⚠️ Errores Comunes

### ❌ Error 1: Copiar sin entender

```
CLAUDE: [Proporciona código]
TÚ: Copias → Pegas → Funciona
RESULTADO: Cuando falla, no sabes qué pasó
```

### ❌ Error 2: Confiar ciegamente

```
CLAUDE: [Código]
TÚ: "Si Claude lo dice, es correcto"
RESULTADO: Bugs en producción
```

### ❌ Error 3: No validar

```
CLAUDE: [Código]
TÚ: [Lo usas sin probar]
RESULTADO: No funciona como esperas
```

### ✅ Forma Correcta

```
CLAUDE: [Código + Explicación]
TÚ: 1. Lo pruebo
    2. Lo entiendo
    3. Lo modifico
    4. Lo valido
    5. Hago más preguntas
```
